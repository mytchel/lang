# A Language

Some kind of a language.

It compiles into a kind of in memory byte code which is then interpretted. 

Check `test.s` for an example of code.

Supports:

* Variables
* Funcitons
* If statements.
* Arguments and return values.
* Special print function.
* Recursion.
* Type annotations.
* Types.
